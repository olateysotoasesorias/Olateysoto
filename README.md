# Olate & Soto Web

Sitio web corporativo construido con Next.js, TypeScript y Tailwind CSS.

## Requisitos

- Node.js instalado
- `npm` disponible en el PATH

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre http://localhost:3000 en tu navegador.

## Notas

- El proyecto ya incluye `app/page.tsx`, `app/layout.tsx`, `app/globals.css`, `tailwind.config.ts` y `components/Navbar.tsx`.
- Si necesitas ajustar la paleta, puedes editar `tailwind.config.ts`.
